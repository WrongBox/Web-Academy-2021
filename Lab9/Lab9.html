What is Inline CSS?

Inline CSS is used for a single HTML element, it uses the style attribute.

What is Internal CSS?

Internal CSS is used for a single HTML page, it has to be used in the head section , that has a style element.

What is External CSS?

External CSS is used for HTML pages, to be able to use it , you would have to add a link to the head section. The file cannot have HTML code, and it will have to be saved using css extension.
